# CodeVibe

CodeVibe is a web-based coding practice platform inspired by FreeCodeCamp.  
It helps users learn programming through structured tasks, real-time coding practice, and instant evaluation.

The platform removes the need for local environment setup by providing in-browser compilers for web-based languages.

---

## What CodeVibe Does

CodeVibe provides an interactive learning experience where users can:

- Practice coding directly in the browser  
- Run code without setting up any environment  
- Solve structured programming problems  
- Follow step-by-step instructions  
- Track personal progress  
- Improve through continuous practice  

---

## Features

### Coding Platform
- Browser-based code editor  
- Inbuilt compiler for supported languages  
- Instant output execution  

### Evaluation System
- Automatic checking of solutions  
- Scoring based on correctness  
- Full marks (100) for correct solutions  
- Marks deduction for incorrect attempts (as per rules)  

### Learning System
- Step-by-step task instructions  
- Beginner-friendly structured flow  
- Focus on logic building and fundamentals  

### Progress Tracking
- Track completed tasks  
- Monitor learning progress  
- View performance history  

### Final Assessment
- MCQ-based final test  
- Overall knowledge evaluation  

### Certificate System
- Certificate awarded after completion  
- Based on performance and test results  

### Feedback System
- Detailed student progress report  
- Highlights strengths and weak areas
